=================================================
Qiskit Mori: A Qiskit Provider for Rigetti Forest
=================================================

This project strives to be a bridge to enable using Qiskit with Rigetti's
forest service. It enables users to use the Qiskit project with Riggetti's
service if they desire. It proves a Qiskit Terra backend provider for Rigetti
backends that can be used like any other Qiskit backend provider.
